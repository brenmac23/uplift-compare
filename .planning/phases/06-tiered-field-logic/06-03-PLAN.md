**generateTier3 signature:**
```typescript
function generateTier3(
  rand: () => number,
  tier1: Tier1Fields,
  tier2: Tier2Fields,
  tierConfig: BudgetTierConfig,
  qnzpe: number
): Tier3Fields
```

**Algorithm:**

1. **Sample ambition target:** Use `weightedSelect(rand(), AMBITION_TARGET_WEIGHTS)` to get an index 0-5, map to values [40, 41, 42, 43, 44, 45]. This is the project's personal goal.

2. **Compute current score:** Build a partial `ProjectInputs` from tier1 + tier2 fields (fill Tier 3 fields with defaults: all false/0/none). Call `scoreExisting()` to get `currentScore`.

3. **Compute gap:** `gap = ambitionTarget - currentScore`. If gap <= 0, the project already meets its target from Tier 1+2 alone. Skip to step 6.

4. **Greedy selection from TIER3_FIELD_COSTS:** Iterate through the cost-ordered options. For each option:
   - If it's a Section E field (`cost === 'expensive'`): skip unless `qnzpe >= 100_000_000` AND all cheaper options have been exhausted or selected AND gap > 0.
   - If `gap <= 0`: stop iterating.
   - Use `rand()` to decide whether this producer "bothers" with this criterion. Probability depends on cost: cheap = 0.85, medium = 0.55, expensive = 0.40. This adds realism — not every producer takes every available option.
   - If selected: set the field's value (true for booleans, threshold-meeting count for attachmentCount/internshipCount, appropriate dollar/percent for E2/E3). For attachmentCount: `isHighBudget ? 4 + Math.floor(rand() * 4) : 2 + Math.floor(rand() * 4)`. For internshipCount: similar logic using QNZPE thresholds from spec. For E2: pick from [0.25, 0.5, 1] using `rand()`. For E3: pick from [500_000, 1_000_000, 2_000_000] using `rand()`.
   - After setting a field, recompute score with `scoreExisting()` and update gap.

5. **Section E last resort (per user decision):** If after iterating all cheap and medium options, gap > 0 AND qnzpe >= 100_000_000, iterate through expensive options (E1, E2, E3). Apply the same `rand()` probability check (0.40). Stop when gap <= 0 or options exhausted.

**CRITICAL — PRNG consumption:** Tier 3 is inherently branch-dependent (some projects chase points, others don't). To maintain PRNG stability:
   - Always consume exactly 1 `rand()` call for the ambition target.
   - For the greedy loop: consume 1 `rand()` call per TIER3_FIELD_COSTS entry regardless of whether it's selected or skipped. If the field needs a value (attachmentCount, internshipCount, E2, E3), consume an additional `rand()` call regardless of whether the field is active. This means the total Tier 3 PRNG consumption is: 1 (ambition) + N (one per option) + M (one per option needing a value), where N and M are fixed constants from TIER3_FIELD_COSTS length.
   - Count the exact PRNG calls in TIER3_FIELD_COSTS and document the total in a comment.

6. **Proposed-only fields:** These are not scored by scoreExisting but must be populated for the proposed engine. Generate them with simple probabilities at the end, AFTER the greedy loop:
   - `hasIndustrySeminars = rand() < 0.5`
   - `hasNZPremiere = rand() < 0.4`
   - `hasIntlPromotion = rand() < 0.4`
   - `hasLocationAnnouncement = rand() < 0.5`
   Always consume exactly 4 `rand()` calls here.

Export `Tier3Fields` and `generateTier3`.

export function generateProject(
  rand: () => number,
  index: number,
  name: string,
  productionType: ProductionType,
  tierConfig: BudgetTierConfig
): { project: Project; existingScore: number } {
  const qnzpe = sampleQnzpe(tierConfig, rand);

  // Tier 1: Fundamentals — budget-correlated base fields
  const tier1 = generateTier1(rand, tierConfig, productionType);

  // Tier 2: Less Fundamental — cross-field correlations from Tier 1
  const tier2 = generateTier2(rand, tier1, tierConfig);

  // Tier 3: Point-chasing — score-gap greedy over cheap criteria
  const tier3 = generateTier3(rand, tier1, tier2, tierConfig, qnzpe);

  // Assemble ProjectInputs from all three tiers
  const inputs: ProjectInputs = {
    projectName: name,
    productionType,
    qnzpe,
    // Tier 1
    hasSustainabilityPlan: tier1.hasSustainabilityPlan,
    hasPreviousQNZPE: tier1.hasPreviousQNZPE,
    hasAssociatedContent: tier1.hasAssociatedContent,
    shootingNZPercent: tier1.shootingNZPercent,
    atlCount: tier1.atlCount,
    hasLeadCast: tier1.hasLeadCast,
    castingLevel: tier1.castingLevel,
    // Tier 2
    hasStudioLease: tier2.hasStudioLease,
    regionalPercent: tier2.regionalPercent,
    picturePostPercent: tier2.picturePostPercent,
    soundPostPercent: tier2.soundPostPercent,
    vfxPercent: tier2.vfxPercent,
    conceptPhysicalPercent: tier2.conceptPhysicalPercent,
    castPercent: tier2.castPercent,
    crewPercent: tier2.crewPercent,
    btlKeyCount: tier2.btlKeyCount,
    btlAdditionalCount: tier2.btlAdditionalCount,
    supportingCastCount: tier2.supportingCastCount,
    // Tier 3
    hasSustainabilityOfficer: tier3.hasSustainabilityOfficer,
    hasCarbonReview: tier3.hasCarbonReview,
    hasMasterclass: tier3.hasMasterclass,
    hasEdSeminars: tier3.hasEdSeminars,
    attachmentCount: tier3.attachmentCount,
    internshipCount: tier3.internshipCount,
    hasKnowledgeTransfer: tier3.hasKnowledgeTransfer,
    commercialAgreementPercent: tier3.commercialAgreementPercent,
    infrastructureInvestment: tier3.infrastructureInvestment,
    premiereType: tier3.premiereType,
    hasFilmMarketing: tier3.hasFilmMarketing,
    hasTourismMarketing: tier3.hasTourismMarketing,
    hasTourismPartnership: tier3.hasTourismPartnership,
    // Proposed-only (from Tier 3)
    hasIndustrySeminars: tier3.hasIndustrySeminars,
    hasNZPremiere: tier3.hasNZPremiere,
    hasIntlPromotion: tier3.hasIntlPromotion,
    hasLocationAnnouncement: tier3.hasLocationAnnouncement,
    // Fixed (Maori fields — Phase 7 handles the special scenario)
    maoriCrewPercent: 0,
    hasLeadCastMaori: false,
  };

  const scoringResult = scoreExisting(inputs);

  const project: Project = {
    id: `seed-${String(index + 1).padStart(3, '0')}`,
    isSeeded: true,
    createdAt: '2026-01-01T00:00:00.000Z',
    inputs,
  };

  return { project, existingScore: scoringResult.totalPoints };
}
