# Uplift Compare

In this folder I have two documents.

* Existing_Production_Rebate_5_Uplift_points_test
* Proposed_Production_Rebate_5_Uplift_points_test

They related to the New Zealand Screen Production Rebate 5% Uplift. One is the existing rules and the other is new proposed rules. I want to build an app that does a comparision to see how different projects would fare between these rules. 

So please devise a system where we can quickly tick checkboxes or enter in information that then feeds into both point systems. Try to do the minimum required information required to accurately fill both out. 

I'd like a screen where you can see each project in detail with a dropdown box to switch between them.

I'd also like to see a summary screen with a description on each project and whether they pass or not each criteria.

Please also seed in 50 fictional projects (a mix of films and tv projects).

Note: this is an international rebate so avoid using names that reference New Zealand or existing franchises. Only generate fake titles.

Half should pass the existing test and half should fail. 

Half should be over $100m QNZPE. All should be over $20m QNZPE.

For the existing test: all should have A1, and none should have C3 and C10. B1 is very rare. Big budget projects will tend to score low in qualifying person questions as they tend to import overseas talent. Section E is rare as it is expensive so the domain of big budget projects. Most projects will reach 80% on C2 especially smaller budget projects.

I will like to deploy on Netlify.

It should be a light theme and if we can use the frontend design plugin to make for an aesthic but tidy look.

It'll be great to be able to export the variable information into an Excel format.

Users should be able to generate new projects.
